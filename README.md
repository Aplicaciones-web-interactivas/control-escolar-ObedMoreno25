# Sistema de Control Escolar

Proyecto desarrollado con Laravel para la administración de un sistema básico de control escolar.

El sistema permite gestionar materias, horarios y grupos dentro de una institución educativa.

---

## Tecnologías utilizadas

- Laravel
- PHP
- MySQL
- Tailwind CSS
- Font Awesome

---

## Funcionalidades

El sistema cuenta con los siguientes módulos:

### Materias
Permite administrar las materias disponibles en el sistema.

- Crear materias
- Editar materias
- Eliminar materias
- Listar materias

---

### Horarios
Permite asignar horarios a las materias y maestros.

Cada horario incluye:

- Materia
- Maestro
- Días de la semana
- Hora de inicio
- Hora de fin

---

### Grupos
Permite crear grupos asociados a un horario específico.

Cada grupo representa una clase disponible para que posteriormente los alumnos puedan inscribirse.
